# The Testing Table

I’m a huge science nerd who runs tons of experements, and I needed a way to organize and light those experements.
This table that is currently built, alongside a breadboard complex I’m designing along side it let me run experements and shoot video of them too!
Currently, it is a small experiment table made from scrap wood with NeoPixel lights added so I can actually see what I’m doing and record videos of tests without everything looking terrible.

## What it does

* gives me a place to run experiments
* holds/organizes random science stuff
* uses NeoPixel lights for better lighting
* helps with filming experiments
* will work with a breadboard/electronics setup I’m designing next to it

## Why I made it

I kept doing experiments in random places, which is not great. I wanted one dedicated way I could build stuff, test stuff, light it properly, and record it.

It is not super built up right now, but it works, and it gives me a better base for future experiments.
